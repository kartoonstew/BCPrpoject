# Black Company Compendium

A Python command-line reference tool for the Black Company tabletop RPG setting.

## What it does
Interactive lookup tool for players and DMs. Browse subclasses, traits, quirks, 
the Black Banner, morale rules, and commissary rules from the Black Company 
Player Handbook.

## How to use
```
python BC.py
```
Follow the prompts to navigate the compendium.

## Contents
- Black Company — Brother creation, traits, and quirks
- Subclasses — All nine Fighter subclasses with level features
- Black Banner — Rules, features, and auras
- Morale — Enemy morale check rules
- Commissary — Magic item durability rules

## Built with
- Python 3
- JSON
